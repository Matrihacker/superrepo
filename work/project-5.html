Cat Simulator

Tensorflow AI image recognition project
Project Overview
The device utilized the Ai Object Detection technology via a camera to drive the car for mimic the behavior of cats. In the night, the cats usually are excited, so when it detect there is human in the vision. It will start to chase the person, like a cat going to rub people’s leg. If there is not person in range, it will looking around to find a people. In the day, the cats are become lazy, so if there is a person approach, it will start to run away from you. 
Devices
Raspi OS, Motor Driver, Pi Cam
The Interaction Flow
Apr 2014 - Mar 2015
The machine can mimic the behavior of a cat, find its owner, and follow automatically after the camera detects her. The device uses Ai object detection technology to drive the robot to mimic the behavior of a cat via the camera.
At night, cats are usually very excited, so when they detect a human in their field of view, they will start chasing him and rubbing his legs. If there is no one in range, it will look around for her. In the daytime, cats become lazy, so they will start running away from you if a human comes near.
